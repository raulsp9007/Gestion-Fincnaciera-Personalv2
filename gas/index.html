💰 CashMap v2 — Google Apps Script

Script para desplegar como web app en Google Apps Script

Pasos para desplegar

  1. Abre script.google.com y crea un nuevo proyecto
  2. Borra el contenido de Code.gs y pega el código de abajo
  3. Guarda (Ctrl+S)
  4. Clic en Implementar → Nueva implementación
  5. Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquier usuario
  6. Copia la URL /exec que aparece
  7. En CashMap v2 → Admin → Conexión GAS → pega la URL → Guardar → Probar conexión
